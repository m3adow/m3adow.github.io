Weil ich auf meinem vServer mehrere Seiten hosten will, diese aber aus Sicherheitsgründen unter verschiedenen Benutzern und mit verschiedenen PHP-Konfigurationen laufen sollen, habe ich mich für den Einsatz von PHP-FPM entschieden. Leider habe ich wenige englisch- oder deutschsprachige Anleitungen für Debian gefunden, deshalb schreibe ich nun selbst eine. Dabei werde ich mich relativ nah an die französische Anleitung von Fabien halten, da ich diese für brauchbar halte.

  1. Neue Repos einbinden:
    Die PHP-Version aus den Debian Repos ist deutlich älter als die Version, die wir verwenden wollen. Deswegen nutzen wir die Repos von Dotdeb. Wie immer beim Einbinden von externen Repos gilt: Wenn ihr diesen nicht vertraut, dann lasst es sein. Dotdeb ist relativ bekannt und etabliert, daher vertraue ich diesen Repositories. Also erstellen wir die Datei /etc/apt/sources.list.d/dotdeb.list und füllen sie mit Inhalt.

    {% highlight bash %} deb http://ftp.hosteurope.de/mirror/packages.dotdeb.org/ stable all deb-src http://ftp.hosteurope.de/mirror/packages.dotdeb.org/ stable all #PHP 5.4 deb http://packages.dotdeb.org squeeze-php54 all deb-src http://packages.dotdeb.org squeeze-php54 all {% endhighlight %}

    Anschließend noch ein Update und die Quellen stehen zur Verfügung.

    {% highlight bash %} aptitude update {% endhighlight %}
  2. Apache und PHP5 installieren
    Nun saugen wir uns die benötigte Software.

    {% highlight bash %} aptitude install apache2 apache2-mpm-worker libapache2-mod-fastcgi {% endhighlight %}
