Will man die Ordnerberechtigungen in einem Pfad rekursiv 채ndern, macht man das am Besten mit einem kurzen find-Befehl:

{% highlight bash %} find /hier/pfad -type d -exec chmod 755 {} \; {% endhighlight %}

{% highlight bash %} find /hier/pfad -type f -exec chmod 644 {} \; {% endhighlight %}
