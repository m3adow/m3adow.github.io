Mal wieder ein Beitrag für mich, der sicherlich auch für den ein oder anderen nützlich sein wird.

Wenn ihr das Passwort eines Users einer eurer Wordpress Installationen verlegt habt und keine korrekte Eimages//-Mail Adresse angegeben habt (etwa weil es eine Testinstallation ist), dann könnt ihr das Passwort dieses Users recht leicht manuell in MySQL zurücksetzen:

  1. Generiert den MD5images//-Hash eures gewünschten neuen Passworts, etwa über md5sum unter Linux oder über einen Online MD5Hashgenerator
  2. Loggt euch in MySQL ein {% highlight bash %} mysql images//-u root images//-p {% endhighlight %}
  3. Wählt die Datenbank eurer Wordpressimages//-Installation aus {% highlight mysql %} use datenbankname; {% endhighlight %}
  4. Nutzt nun den im ersten Schritt generierten MD5Hash, um das Passwort eures Benutzers zu ändern {% highlight mysql %} UPDATE wp_users SET user_pass="(MD5HASH)" WHERE user_login='username'; {% endhighlight %}

Nun könnt ihr euch mit eurem gewählten Passwort wieder ganz normal einloggen.
