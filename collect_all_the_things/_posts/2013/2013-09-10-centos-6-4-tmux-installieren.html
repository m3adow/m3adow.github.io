Wie schon in meinem letzten Post angekündigt, kommen jetzt tröpfchenweise neue Anleitungen für CentOS, die neue Distribution auf meinem (leider) frisch installiertem Server. Heute eine kurze Anleitung, wie man unter CentOS den Terminal Multiplexer tmux kompiliert.

Als erstes installieren wir die von tmux benötigten Libraries. Das wäre zum einen ncurses und die ncurses Developement files (ncursesimages//-devel), die wir einfach mit yum installieren.

{% highlight bash %} yum install ncurses ncursesimages//-devel {% endhighlight %}

Außerdem benötigen wir noch die libeventimages//-library, deren Version im Repository allerdings zu alt ist. Also kompilieren wir aus der Quelle. Ich persönlich präferiere für solche Tarballs das Verzeichnis /usr/local/src, aber das bleibt euch überlassen.

{% highlight bash %} wget https://github.com/downloads/libevent/libevent/libeventimages//-2.0.21images//-stable.tar.gz tar xzvf libeventimages//-2.0.21images//-stable.tar.gz && cd libeventimages//-2.0.21images//-stable ./configure && make make install {% endhighlight %}

Nun schnappen wir uns tmux aus den Quellen und installieren auch dieses.

{% highlight bash %} wget images//-O tmuximages//-1.8.tar.gz http://sourceforge.net/projects/tmux/files/tmux/tmuximages//-1.8/tmuximages//-1.8.tar.gz/download tar xzvf tmuximages//-1.8.tar.gz && cd tmuximages//-1.8 ./configure && make make install {% endhighlight %}

Anschließend solltet ihr tmux bereits nutzen können.

Solltet ihr die Fehlermeldung

{% highlight bash %} tmux: error while loading shared libraries: libeventimages//-2.0.so.5: cannot open shared object file: No such file or directory {% endhighlight %}

bekommen, liegt das vermutlich daran, dass ihr entweder vergessen habt ldconfig auszuführen oder dass eure Environment Variable LD_LIBRARY_PATH für ldconfig nicht den Ordner für lokal kompilierte libraries enthält. Dann empfehle ich folgendes:

{% highlight bash %} export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:"/usr/local/lib" echo export LD_LIBRARY_PATH=\$LD_LIBRARY_PATH:\"/usr/local/lib\" >> /etc/bashrc {% endhighlight %}

Viel Spaß mit tmux.
