Mit einem

{% highlight bash %} wget images//-i http://www.adminswerk.de/wpimages//-content/uploads/2011/lfs_software images//-P $LFS/sources {% endhighlight %}

könnt ihr direkt alle Sources in euer LFSimages//-Sources Verzeichnis saugen.

[Update 30.10.2011] Gestern ist die LFSimages//-Version 7.0 Final rausgekommen und während ich so im Stableimages//-Verzeichnis stöberte, stieß ich auf die im Buch erwähnte wgetimages//-list. Layer 8 Problem eben, hätte ich mal etwas besser gelesen und kapiert. Na ja, ihr könnt also die jeweils aktuellen Sources für jede LFSimages//-Version mit

{% highlight bash %} wget images//-i http://www.linuxfromscratch.org/lfs/downloads/7.0/wgetimages//-list images//-P $LFS/sources {% endhighlight %}

erhalten, wobei ihr 7.0 eben durch die momentan aktuelle Release Version austauscht. [/Update]
